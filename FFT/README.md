# Geração das Funções Espectrais

Antes de executar o script deste diretório siga [estas instruções](https://github.com/JaoIndio/Dissertacao_Material_Complementar/tree/master/simulacoes/codigos) para configurar seu ambiente.

Aqui encontra-se uma grande variedade de coletas que foram realizadas durante este trabalho. Há dados extraídos da fonte de LED e do diodo Laser.

Ao executar o script `FFT.py` já estará selecionado o arquivo CSV correspondente às imagens (`acquisition_257.csv`) mostradas no texto da dissertação.

Para analisar os dados apenas precione o botão **Carregar CSV**. Em seguida, de zoom in na região de interesse para ser analisada, precione; com o mouse no gráfico, o botão esquerdo do mouse e arraste. Isso irá definir o tamanho da janela a ser analisada. Por fim aperte o botão **FFT**. Após o processamento você pode arrastar a janela vermelha (não precione nas bordas, isso irá mudar o tamanho dela e uma nova FFT terá que ser realizada) para observar a FFT de outras regiões.
